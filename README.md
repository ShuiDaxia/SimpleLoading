# SimpleLoading

更方便的使用Loading,集成了AVLoadingIndicatorView 



## Step 1.

在项目根build.gradle下添加
```
allprojects {
    	repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```


## Step 2.

在项目根build.gradle下添加
```
dependencies {
    compile 'com.github.diswy:SimpleLoading:v1.0'
}
```
